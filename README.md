# Fire_Department_SQL_Server
Snohomish Fire Department 4 SQL Server Repository

## Functions

* Func1 - Total Calls for Service in District 
* Func2 - Total Responses 
* Func3 - Total Responses in District 
* Func4 - Total Calls for Service with Auto Aid
* Func5 - AutoAid Out Responses
* Func6 - Total Calls for Service
* Func7 - Response Time(Priority 1, 2, 3, 4)
* Func8 - Total FD4 Responses in District
* Func9 - Auto Aid In
* Func10 - Total FD4 Calls for Service in District
* ERF - When the 16th/22nd/7th/19th Firefighter arrives on scene and is depended on call type and urban/suburban
* TravelTimes - How long it takes from unit leaves to arrives on scene(90th percentile)
* CallProcessTime - period of time between call created and the stations get the call(90th percentile)
* TurnoutTime - Period of time recorded from dispact to first unit enroute(90th percentile)
* Creat_Unit_Staffing_Table  - creates table with unit to personal equivalent. Used for ERF
* Load_excel_into_sql - Converts Excel to SQL orders of magnitude faster than data wizard
